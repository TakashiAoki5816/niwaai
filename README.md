# 庭愛（にわあい）
あなたが好きな日本庭園を見つけるをコンセプトにした共有サイトです。
「投稿の閲覧や投稿」、「投稿にいいねやコメント機能」、「気に入ったユーザーのフォロー機能」などを通じて、「あなたの好きな日本庭園に出会うツール」としてご利用いただけます。
URL: https://www.niwaai.com
